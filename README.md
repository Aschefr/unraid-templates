# Unraid Templates

This repository contains the official Unraid Community Applications XML templates for my applications.

## Included Templates

* **Log-to-LLM Sentinel**: AI-powered log monitoring and analysis tool.
* **Alanbix**: Custom application.
* **OmniBank**: Custom banking application.

## How to add this repository to Unraid

To test these templates locally before they are officially approved by Community Applications:
1. Go to the **Docker** tab in Unraid.
2. At the bottom, click on **Add Container**.
3. Under **Template**, you can select "User Templates" or paste the RAW XML url of one of these files.

*(Once added to the official Community Applications store, these apps will be searchable directly from the "Apps" tab).*
