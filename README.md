# Unraid Templates - A.SCH3

This repository contains the Unraid Community Applications (CA) XML templates for dockerized applications developed by **A.SCH3** (Aschefr).

---

## 🚀 Applications List

### 🎮 [Alanbix](https://github.com/Aschefr/Alanbix)
* **Description (EN):** Platform to manage tournaments and interact with AIs. Features a rich interface with player management, game tracking, and integrated AI chat via Ollama. *(Note: The application interface is currently available in French only).*
* **Description (FR):** Plateforme pour gérer des tournois et interagir avec des IA. Interface riche avec gestion des joueurs, des parties, et chat IA intégré via Ollama. *(Note : L'interface de l'application est uniquement disponible en français).*
* **Unraid Template:** [alanbix.xml](https://github.com/Aschefr/unraid-templates/blob/master/alanbix.xml)

### 🏦 [OmniBank-Local](https://github.com/Aschefr/OmniBank-Local)
* **Description (EN):** Financial backend initially designed for desktop, adapted to run autonomously on Unraid via Docker.
* **Description (FR):** Backend financier conçu initialement pour le desktop, adapté pour fonctionner en autonomie sur Unraid via Docker.
* **Unraid Template:** [omnibank-local.xml](https://github.com/Aschefr/unraid-templates/blob/master/omnibank-local.xml)

### 🛡️ [Log-to-LLM Sentinel](https://github.com/Aschefr/log-to-llm-sentinel)
* **Description (EN):** AI-powered log monitoring and analysis tool. Watches logs and automatically uses local AI (Ollama) to detect, analyze, and alert you of anomalies in real-time.
* **Description (FR):** Outil de surveillance et d'analyse de journaux (logs) propulsé par l'IA. Surveille vos logs et utilise automatiquement une IA locale (Ollama) pour détecter, analyser et alerter des anomalies en temps réel.
* **Unraid Template:** [log-to-llm-sentinel.xml](https://github.com/Aschefr/unraid-templates/blob/master/log-to-llm-sentinel.xml)

## 🛠️ Installation on Unraid
Since this repository is officially approved on the **Unraid Community Applications** store, you can simply:
1. Open your Unraid web GUI.
2. Go to the **Apps** tab.
3. Search for **Alanbix**, **OmniBank**, or **Sentinel** (or filter by developer **Asch3**).
4. Click **Install**.
